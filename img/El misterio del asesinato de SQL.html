Misterio de asesinato de SQL

¿Puedes averiguar quién es?

¡Ha habido un asesinato en SQL City! The SQL Murder Mystery está diseñado para ser tanto una lección autodirigida para aprender conceptos y comandos de SQL como un juego divertido para que los usuarios experimentados de SQL resuelvan un crimen intrigante.

¿Eres nuevo en SQL?

Este ejercicio está pensado más como una forma de practicar las habilidades de SQL que como un tutorial completo. Si nunca has usado SQL, Pruebe el tutorial. Si realmente quieres aprender mucho sobre SQL, es posible que prefieras un tutorial como Seleccione Star SQL.

Si se siente cómodo con SQL, puede Sumérgete a continuación!

Los detectives de SQL experimentados comienzan aquí

Ha ocurrido un crimen y el detective necesita tu ayuda. El detective te dio el informe de la escena del crimen, pero de alguna manera lo perdiste. Recuerda vagamente que el crimen fue un asesinato que ocurrió en algún momento del 15 de enero de 2018 y que tuvo lugar en SQL City. Comience por recuperar el informe correspondiente de la escena del crimen de la base de datos del departamento de policía.

Exploración de la estructura de la base de datos

Los usuarios experimentados de SQL a menudo pueden usar consultas de base de datos para inferir la estructura de una base de datos. Pero cada sistema de base de datos tiene diferentes formas de gestionar esta información. SQL Murder Mystery se construye con SQLite.
